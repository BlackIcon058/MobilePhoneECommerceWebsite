# MobilePhoneECommerceWebsite
 MobilePhoneECommerceWebsite Laravel
Link website: http://quanhoangmobileshop.infinityfreeapp.com/

### Website Features
**User**
- View product details
- Login, registration
- Product search
- Place order, make payment
- View purchase order

**Admin**
- Login, registration
- Statistics and reporting
- Product management
- User management
- Customer management
- Order management

### Installing
Run on Xampp

- [1] Gitbash: git clone https://github.com/BlackIcon058/MobilePhoneECommerceWebsite.git
- [2] Move folder in htdocs
- [3] Add DB file (DBMobileShop.sql) on http://localhost/phpmyadmin/ with name: elaravel
- [4] run http://localhost/MobilePhoneECommerceWebsite/
