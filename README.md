# MobilePhoneECommerceWebsite
 MobilePhoneECommerceWebsite Laravel
Link website: http://quanhoangmobileshop.infinityfreeapp.com/

Link website admin: http://quanhoangmobileshop.infinityfreeapp.com/admin

### Website Features
**User**
- View product details
- Login, registration
- Product search
- Place order, make payment
- View purchase order

**Admin**
- Login, registration
- Statistics and reporting
- Product management
- User management
- Customer management
- Order management

### Installing
Run on Xampp

- [1] Gitbash: git clone https://github.com/BlackIcon058/MobilePhoneECommerceWebsite.git
- [2] Add DB file
